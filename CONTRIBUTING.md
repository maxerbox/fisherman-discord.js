# Contributing

* Please respect the code style

## Building docs

I'm using docma to build the docs. It can be installed with:

```terminal
npm i docma -g
```

Docma: [https://github.com/onury/docma](https://github.com/onury/docma)

 to build the docs use:

 ```terminal
 docma -c docma.config.json
 ```
